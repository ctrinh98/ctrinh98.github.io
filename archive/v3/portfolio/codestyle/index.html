CodeStyle

>   Read your code better than ever before.

Dev summary

CodeStyle is a web application where users can enter in their Python code and can receive personalized suggestions on how to make their code more readable for themselves and for everyone else.

The goal of the application is to teaching budding programmers what it means to have good coding style by giving them suggestions that improves their code formatting and hopefully puts them in a mindset that allows them to recognize good coding style skills.

CodeStyle is developed under the updog team within the Boston University Spark! Innovation Fellowship over a period of four months. Mentors from Liberty Mutual and Boston University Information Services & Technology worked with the updog team to help guide the creation of the app. Check out the app at the link below!
